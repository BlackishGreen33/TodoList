## 🧭 你可以做什么

- 添加待办事项
- 标记单项或全部待办为完成
- 双击编辑待办内容
- 按全部 / 未完成 / 已完成筛选
- 清除已完成待办
- 使用 `localStorage` 保存数据

## 🛠 使用技术

- React 19
- Vite 8
- nanoid

## 🚀 本地运行

```bash
npm install
npm run dev
```

打开 Vite 输出的本地地址即可预览，通常是：

```text
http://127.0.0.1:5173/
```

## 📦 构建

```bash
npm run build
```
